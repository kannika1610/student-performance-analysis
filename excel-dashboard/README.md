## 📊 Excel Dashboard – Student Performance

### 📌 Overview  
This project uses Excel to analyze student performance based on study habits, sleep, and attendance.

### 🛠️ Tools Used  
- Microsoft Excel  
- Pivot Tables  
- Charts  
- Slicers  

### 📊 Features  
- Interactive dashboard  
- Study vs Exam Score analysis  
- Sleep vs Performance analysis  
- Filters for better insights
