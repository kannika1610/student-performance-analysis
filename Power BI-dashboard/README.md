## 📊 Power BI Dashboard – Student Performance

### 📌 Overview  
This project uses Power BI to create an interactive dashboard to analyze student performance.

### 🛠️ Tools Used  
- Power BI  

### 📊 Features  
- KPI cards (average score, study, sleep)  
- Interactive slicers  
- Visual charts for performance analysis  

### 🔍 Insight  
Higher study time and balanced lifestyle improve exam scores.
