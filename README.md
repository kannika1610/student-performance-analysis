# 📊 Student Performance Analysis

## 📌 Overview  
This project focuses on analyzing student performance using data visualization techniques in Excel and Power BI. The goal is to understand how different factors such as study hours, sleep patterns, attendance, and lifestyle habits impact exam scores.

This project demonstrates the use of data analysis and visualization tools to derive meaningful insights and support better decision-making.

---

## 🎯 Objectives  
- Analyze student performance based on multiple factors  
- Identify patterns affecting exam scores  
- Build interactive dashboards for better understanding  
- Present insights in a clear and visual format  

---

## 🛠️ Tools & Technologies  
- Microsoft Excel  
- Power BI  
- Data Cleaning  
- Data Visualization

## 📊 Dataset Description  
The dataset contains information related to student behavior and performance, including:

- Study Hours per Day  
- Sleep Hours  
- Social Media Usage  
- Attendance Percentage  
- Diet Quality  
- Exercise Frequency  
- Mental Health Rating  
- Exam Score  
- Performance Level  

---

## 📈 Dashboards  

### 🔹 Excel Dashboard  
- Built using pivot tables and charts  
- Interactive slicers for filtering  
- Basic analysis of student performance  

### 🔹 Power BI Dashboard  
- KPI cards for key metrics  
- Advanced visualizations  
- Dynamic filtering using slicers  

---

## 🔍 Key Insights  
- Students who spend more time studying tend to score higher  
- Proper sleep has a positive impact on performance  
- Higher attendance leads to better academic results  
- Excessive social media usage negatively affects exam scores  
- Balanced lifestyle contributes to consistent performance
